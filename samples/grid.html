Grid

The base grid

Grid system largely inspired by Melody, an extremely lightweight CSS grid framework. I've adapted and borrowed elements from a number of other frameworks to evolve this into something that meets my needs, and done things a little differently to make it more flexible.

No floats means no dodgy IE hacks to try and clear them. Just the nice and clean, display: inline-block;

Uses padding and box-sizing: border-box; instead of margin for spacing, which makes it much easier to create a grid-within-a-grid or marginless layouts.

If you need spacing, then style the content; If you need a grid without spacing, then style the grid itself, or use the class .remove-padding.

More fun with the grid

Uses the class .stick-to-grid to preserve the width and offset of grid elements at mobile screen sizes.

Uses the class .offset-n to add left-margin to an element to create an offset. The value of n is the number of columns (1-12) for the offset.

Uses the class .keep-shape to add additional padding to grid elements to keep them at 960-grid size if the grid expands to 1280 wide on large screens.
